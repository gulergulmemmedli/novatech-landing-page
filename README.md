NovaTech — Responsiv Landing Page
Verilmiş dizayna əsasən hazırlanmış, framework istifadə olunmadan yazılmış tam responsiv, semantik və əlçatan landing page.
Haqqında
Layihə 4 bölmədən ibarətdir: Hero, Haqqımızda, Qalereya və Əlaqə formu. Naviqasiya, mobil menyu və form validasiyası Vanilla JavaScript ilə yazılıb.
İstifadə olunan texnologiyalar

* HTML5 (semantik strukturlar)
* CSS3 (Flexbox, Grid, mobile-first responsiv dizayn)
* Vanilla JavaScript (framework istifadə olunmayıb)

Necə işə salmaq olar

1. Repo-nu kompüterinə yüklə (Code → Download ZIP, və ya `git clone`)
2. `index.html` faylını istənilən brauzerdə aç

Backend yoxdur, əlavə quraşdırma tələb olunmur.
Xüsusiyyətlər

* Mobile-first responsiv dizayn (3 breakpoint: mobil / tablet / desktop)
* Klaviatura ilə tam naviqasiya, görünən fokus stilləri
* Mobil hamburger menyu
* Nav linklərində smooth scroll
* Əlaqə formunda client-side validasiya (boş sahə, email formatı)
* ARIA atributları və semantik HTML ilə əlçatanlıq
